# Olivier Reynes Photography — Website

Site vitrine professionnel pour Olivier Reynes, photographe.

## Stack

- **Framework** : Next.js 15 (App Router)
- **Style** : Tailwind CSS 4 + Design System custom
- **DB** : PostgreSQL (Neon) + Prisma
- **Auth** : Auth.js v5
- **Images** : Cloudinary
- **Paiements** : Stripe
- **Déploiement** : Vercel

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
